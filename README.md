# Customer Churn Prediction System

An end-to-end Machine Learning web application that predicts customer churn
using a trained classification model.

## Tech Stack
- Python
- FastAPI (Backend)
- Streamlit (Frontend)
- Scikit-learn
- Pandas

## Features
- Train ML model using CSV upload
- Test model performance
- Predict customer churn via user input
- Clean and user-friendly UI

## How to Run

### Backend
```bash
cd backend
uvicorn main:app --reload
